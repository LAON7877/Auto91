## 系統概述

Auto91 是一個支援雙系統的自動交易平台，整合TX（永豐期貨）和BTC（幣安期貨）交易系統。TX系統支援大台指、小台指和微台指的智能交易，BTC系統支援BTCUSDT永續合約交易，並整合了Cloudflare Tunnel全自動管理和動態端口控制。

### 雙系統架構 (v1.4.0)
- **TX系統**：基於永豐API的台指期貨自動交易（TXF、MXF、TMF）
- **BTC系統**：基於幣安API的加密貨幣期貨自動交易（BTCUSDT永續合約）
- **統一管理**：兩個系統功能完全對等，但數據和配置完全分離
- **智能路由**：支援自動識別TradingView訊號並路由到對應系統

### 最新更新 (v1.3.5)
- **交易記錄持久化系統**：新增 JSON 檔案儲存機制，確保所有交易參數完整保存到 `transdata/` 目錄
- **通知一致性修復**：解決手動平倉通知顯示錯誤的問題，確保提交和成交通知內容一致
- **訂單映射增強**：改進 `order_callback` 函數的推斷邏輯，提升通知準確性
- **自動清理機制**：自動保留最近 30 天的交易記錄，避免磁碟空間浪費
- **動作顯示統一**：新增 `get_action_display_by_rule()` 函數，統一動作顯示邏輯

### 重要更新 (v1.3.4)
- **永豐手動下單參數格式標準化**：改為使用永豐官方參數格式，不再依賴中文 direction 參數

## 核心功能

### TX系統（永豐期貨）
- **智能交易日判斷**：支援期貨週六夜盤交易，正確處理民國年假期檔案
- **多合約交易**：支援 TXF（大台）、MXF（小台）、TMF（微台）
- **永豐API整合**：完整的期貨交易API支援，包含開平倉、查詢、風控
- **交易記錄追蹤**：30天歷史交易記錄，自動生成日報月報

### BTC系統（幣安期貨）
- **加密貨幣期貨**：支援BTCUSDT永續合約交易
- **幣安API整合**：完整的期貨交易API，HMAC-SHA256簽名驗證
- **槓桿交易**：支援5x槓桿，動態倉位計算和風險管理
- **24/7交易**：支援加密貨幣市場全天候交易

### 統一功能
- **動態端口控制**：通過 `port.txt` 靈活配置系統端口
- **Cloudflare Tunnel**：自動隧道管理，取代ngrok
- **三面板Web界面**：設置、TX交易、BTC交易獨立管理
- **雙Telegram通知**：TX和BTC系統獨立通知機制
- **智能Webhook路由**：自動識別TradingView訊號並分發到對應系統
- **完整日誌系統**：分離的系統日誌和請求日誌記錄

## 系統架構

### 端口控制架構
- **統一配置**：使用 `port.txt` 作為端口設置的唯一來源
- **自動配置**：首次啟動時自動生成配置文件，預設端口 5000
- **錯誤處理**：完善的端口讀取錯誤處理和預設值機制

### 合約資訊管理
- **即時更新**：一鍵重新整理合約相關資訊
- **視覺反饋**：載入動畫提供更好的操作體驗
- **統一介面**：與其他功能區域保持一致的視覺和交互設計

### ngrok 自動管理架構
- **自動啟動**：程式啟動時自動啟動 ngrok 背景進程
- **自動重啟**：檢測到 ngrok 異常時自動重啟（5秒延遲）
- **自動關閉**：程式退出時自動清理 ngrok 進程
- **自動升級**：背景檢查並支援 ngrok 版本自動更新
- **錯誤恢復**：自動處理 4040 API 錯誤，確保服務穩定

### 交易日判斷架構
- **源頭**：`main.py` 中的 `is_trading_day_advanced()` 函數
- **調用鏈**：`web/main.js` → `/api/trading/status` → `main.py`
- **假期檔案**：支援民國年格式（如 `holidaySchedule_114.csv`）

### 交易時段
- **早盤**：08:45-13:45
- **夜盤**：14:50-次日05:00（週六夜盤到週六凌晨05:00）

### ngrok 管理特點
- **零配置**：用戶無需手動啟動或配置 ngrok
- **高可用性**：自動故障檢測和恢復機制
- **資源管理**：程式退出時自動清理，避免進程殘留
- **版本管理**：支援自動檢查和升級 ngrok 版本
