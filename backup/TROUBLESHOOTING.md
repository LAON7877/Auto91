# 故障排除指南

1. **無法啟動主程式**
   - 檢查 Python 版本與依賴是否安裝齊全
   - 查看終端機錯誤訊息
2. **API 連線失敗**
   - 檢查 API Key、憑證、網路連線
   - 檢查永豐 API 是否維護中
3. **ngrok 無法連線**
   - 檢查 ngrok token 是否正確
   - 檢查防火牆與網路環境
4. **Telegram Bot 無法推播**
   - 檢查 Bot Token 與 Chat ID
   - 確認 Bot 是否啟用
