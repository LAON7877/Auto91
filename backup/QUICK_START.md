# 5 分鐘快速上手 Auto91

1. 下載並安裝 Python 3.8+ 與 pip
2. 下載本專案並安裝依賴：
   ```
   pip install -r requirements.txt
   ```
3. 準備好永豐 API Key、Secret Key、身份證字號、憑證檔案與密碼
4. 啟動主程式：
   ```
   python server/main.py
   ```
5. 用瀏覽器開啟 http://localhost:5000 進行設置與登入
6. 綁定 Telegram Bot，設定通知 ID
7. 開始自動交易與監控

遇到問題請參考 backup/TROUBLESHOOTING.md
