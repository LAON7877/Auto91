# 5 分鐘快速上手 Auto91

## 系統特色
- **一鍵啟動**：雙擊 `快啟.bat` 即可啟動完整系統
- **ngrok 全自動管理**：無需手動啟動、停止或配置 ngrok
- **自動故障恢復**：ngrok 異常時自動重啟，確保服務持續可用
- **完整日誌系統**：前端系統日誌與 ngrok 請求日誌整合

## 快速啟動步驟

### 1. 環境準備
```bash
# 下載並安裝 Python 3.8+ 與 pip
# 下載本專案並安裝依賴：
pip install -r requirements.txt
```

### 2. 配置設定
```bash
# 準備好永豐 API 憑證：
# - API Key
# - Secret Key  
# - 身份證字號
# - 憑證檔案 (.pfx)
# - 憑證密碼

# 準備 Telegram Bot：
# - Bot Token
# - Chat ID
```

### 3. 一鍵啟動
```bash
# 方法一：使用批處理檔案（推薦）
雙擊 快啟.bat

# 方法二：直接執行 Python
python server/main.py
```

### 4. 系統自動初始化
啟動後系統會自動執行：
- ✅ 啟動 Flask Web 服務器（端口 5002）
- ✅ 自動啟動 ngrok 背景進程
- ✅ 開啟 Web 管理介面
- ✅ 初始化交易日判斷系統

### 5. 登入與配置
1. 用瀏覽器開啟 http://localhost:5002
2. 填寫永豐 API 憑證和 Telegram Bot 設定
3. 上傳假期檔案和憑證檔案
4. 點擊「登入」開始使用

## ngrok 自動管理說明

### 無需手動操作
- **自動啟動**：程式啟動時自動啟動 ngrok
- **自動重啟**：檢測異常時自動重啟（5秒延遲）
- **自動關閉**：程式退出時自動清理 ngrok 進程
- **自動升級**：背景檢查並支援版本自動更新

### 狀態監控
- 前端可即時查看 ngrok 狀態
- 支援延遲、連接數、請求日誌監控
- 自動錯誤恢復，無需人工干預

## 系統功能

### 交易日判斷
- 支援期貨週六夜盤交易
- 自動讀取民國年假期檔案
- 統一交易日判斷邏輯

### 交易管理
- 支援大台指、小台指、微台指
- 即時帳戶狀態和持倉監控
- 自動連線時長管理（12小時自動重連）

### 通知系統
- Telegram Bot 即時通知
- 前端系統日誌記錄
- ngrok 請求日誌監控

## 關閉系統

### 正常關閉
- 關閉 Web 介面視窗即可
- 系統會自動執行清理工作：
  - ✅ 停止 ngrok 進程
  - ✅ 登出永豐 API
  - ✅ 清理所有資源
  - ✅ 重置登入狀態

### 緊急關閉
- 使用 Ctrl+C 強制關閉
- 系統仍會嘗試執行清理工作

## 故障排除

### 常見問題
1. **ngrok 無法啟動**：檢查網路連線，系統會自動重試
2. **API 連線失敗**：檢查憑證設定和網路
3. **前端無法訪問**：確認端口 5002 未被占用

### 自動恢復
- 大部分問題會通過自動恢復機制解決
- 如需手動干預，請參考 `backup/TROUBLESHOOTING.md`

## 進階功能

### ngrok 管理 API
```bash
# 檢查 ngrok 狀態
curl http://localhost:5002/api/ngrok/status

# 手動重啟 ngrok
curl -X POST http://localhost:5002/api/ngrok/stop
curl -X POST http://localhost:5002/api/ngrok/start

# 檢查版本更新
curl -X POST http://localhost:5002/api/ngrok/check_update
```

### 系統日誌
- 前端系統日誌：即時顯示系統狀態
- ngrok 請求日誌：監控外部訪問記錄
- 交易日誌：記錄交易相關操作
